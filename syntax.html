Because HTML is often the people’s first experience with coding, there can be a fair amount of apprehension about learning it. Thankfully, HTML syntax is relatively simple and easy to learn.

A markup language

HTML is a markup language. Content on the page is “marked up” by tags which identify the content inside them. 

<p>This is a paragraph.</p>

Basic document structure

Every HTML document contains an <html> element, a <head>, and a <body>.

DOCTYPES

A doctype declaration tells the browser which version of HTML to expect when rendering a page. 

<!DOCTYPE html>

Element attributes

Attributes provide additional information about elements.

Replaced elements

Some elements, like images, are replaced by external resources.

Code structure

HTML documents create structure by nesting elements inside one another properly.

Commenting code

<!-- This is a comment -->

Using special characters

Reserved characters in HTML can be displayed using named character entities like &amp;.
