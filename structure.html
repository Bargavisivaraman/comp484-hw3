Document Structure

Every HTML document begins with a <!DOCTYPE> declaration, followed by the <html> element which contains the entire page.

Inside the <html> element are two primary sections: the <head> and the <body>.

The Head Element

The <head> section contains metadata about the document, such as the page title, character encoding, and links to external resources like stylesheets or scripts.

The Body Element

The <body> contains all visible page content, including headings, paragraphs, images, lists, tables, and links.

Semantic elements such as <header>, <nav>, <main>, <section>, and <footer> help describe the structure and meaning of content.
